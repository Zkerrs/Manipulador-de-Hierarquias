# Screenshots para o README

Coloque aqui as imagens referenciadas em [`../../README.md`](../../README.md).

## Obrigatório para a imagem principal do README

1. Abra `Manipulador Hierarquia/index.html` no navegador.
2. (Opcional) Clique em **Preencher exemplo** e **Aplicar hierarquia** para uma captura consistente.
3. Tire um print da janela (ou use a ferramenta de captura do sistema).
4. Salve como **`tela-principal.png`** nesta pasta (`docs/screenshots/`).

Formatos aceitos no GitHub: **PNG** ou **JPEG**. O README usa `tela-principal.png` — se usar `.jpg`, altere a extensão no `README.md` na raiz.
